# Problem 4: Segment Tree - Range Sum Query

## Problem Statement
Build a segment tree for range sum queries with point updates.

## Input Format
- An array and queries (update, range_sum).

## Output Format
- Integers for range sum queries.

## Constraints
- 1 <= n <= 3*10^4

## Example
**Input:** arr=[1,3,5,7,9,11], sumRange(1,3)  
**Output:** 15
