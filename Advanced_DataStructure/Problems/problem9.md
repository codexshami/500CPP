# Problem 9: Skip List

## Problem Statement
Implement a skip list that supports search, insert, and delete in O(log n) expected time.

## Input Format
- Operations: search(val), insert(val), delete(val).

## Output Format
- Boolean for search.

## Constraints
- 1 <= n <= 10^4

## Example
**Input:** insert(3), insert(7), search(3)  
**Output:** True
