# Problem 20: Monotonic Stack

## Problem Statement
Use a monotonic stack to find the next greater element for each element in an array.

## Input Format
- A list of integers.

## Output Format
- A list where each element is the next greater element, or -1.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** arr = [4, 5, 2, 10, 8]  
**Output:** [5, 10, 10, -1, -1]
