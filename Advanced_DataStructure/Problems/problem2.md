# Problem 2: LRU Cache

## Problem Statement
Design a data structure for Least Recently Used (LRU) Cache that supports get and put in O(1).

## Input Format
- Operations: get(key), put(key, value), capacity.

## Output Format
- Integer for get (-1 if not found).

## Constraints
- 1 <= capacity <= 3000

## Example
**Input:** LRUCache(2), put(1,1), put(2,2), get(1)  
**Output:** 1
