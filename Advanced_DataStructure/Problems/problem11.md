# Problem 11: Order Statistics Tree

## Problem Statement
Implement a BST that supports select(k) — find kth smallest — and rank(x) in O(log n).

## Input Format
- Operations: insert, select(k), rank(x).

## Output Format
- Integer.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** insert(3,1,4,5,2), select(3)  
**Output:** 3
