# Problem 8: Implement Min-Max Heap

## Problem Statement
Implement a data structure that supports O(1) findMin and findMax, and O(log n) insert/delete.

## Input Format
- Operations: insert, findMin, findMax, deleteMin, deleteMax.

## Output Format
- Integers.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** insert(5), insert(3), insert(8), findMin()  
**Output:** 3
