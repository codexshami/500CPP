# Problem 5: Segment Tree - Range Min Query

## Problem Statement
Build a segment tree to answer range minimum queries.

## Input Format
- An array and queries.

## Output Format
- Minimum value in the given range.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** arr=[2,5,1,4,9,3], query(1,4)  
**Output:** 1
