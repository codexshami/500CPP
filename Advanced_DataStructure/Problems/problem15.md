# Problem 15: Treap

## Problem Statement
Implement a Treap (tree + heap) with random priorities for balanced BST operations.

## Input Format
- Operations: insert, delete, search.

## Output Format
- Boolean for search.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** insert(5), insert(3), insert(7), search(3)  
**Output:** True
