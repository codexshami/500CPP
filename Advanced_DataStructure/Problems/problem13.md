# Problem 13: Sparse Table

## Problem Statement
Build a Sparse Table for O(1) range minimum queries after O(N log N) preprocessing.

## Input Format
- An array, range queries.

## Output Format
- Minimum in range.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** arr=[7,2,3,0,5,10,3,12,18], query(0,4)  
**Output:** 0
