# Problem 6: Fenwick Tree (Binary Indexed Tree)

## Problem Statement
Implement a Fenwick tree for prefix sum queries and point updates.

## Input Format
- An array and update/query operations.

## Output Format
- Prefix sums.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** arr=[1,2,3,4,5], prefixSum(3)  
**Output:** 6
