# Problem 19: Red-Black Tree Insert

## Problem Statement
Implement insertion into a Red-Black tree with proper rebalancing.

## Input Format
- Values to insert.

## Output Format
- The tree in-order traversal.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** insert(7,3,18,10,22,8,11)  
**Output:** [3,7,8,10,11,18,22]
