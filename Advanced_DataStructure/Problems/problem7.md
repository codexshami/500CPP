# Problem 7: Disjoint Set Union (Union-Find)

## Problem Statement
Implement Union-Find with path compression and union by rank.

## Input Format
- Operations: union(x,y), find(x), connected(x,y).

## Output Format
- Boolean for connected, integer for find.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** union(1,2), union(2,3), connected(1,3)  
**Output:** True
