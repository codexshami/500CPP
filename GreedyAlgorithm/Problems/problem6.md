# Problem 6: Minimum Number of Coins

## Problem Statement
Given a value `V`, find the minimum number of coins needed to make the change using the Indian denomination system: {1, 2, 5, 10, 20, 50, 100, 200, 500, 2000}.

## Input Format
- An integer `V` — the value to make change for.

## Output Format
- A list of coins used to make the change.

## Constraints
- 1 <= V <= 10^6

## Example
**Input:** V = 93  
**Output:** [50, 20, 20, 2, 1]  
**Explanation:** Minimum 5 coins needed.
