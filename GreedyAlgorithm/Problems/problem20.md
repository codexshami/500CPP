# Problem 20: Smallest Number with Given Digit Sum and Number of Digits

## Problem Statement
Find the smallest number with a given number of digits `m` and digit sum `s`. If no such number exists, return -1.

## Input Format
- An integer `m` — number of digits.
- An integer `s` — required digit sum.

## Output Format
- A string representing the smallest number, or "-1" if not possible.

## Constraints
- 1 <= m <= 100
- 0 <= s <= 900

## Example
**Input:** m = 2, s = 9  
**Output:** "18"  
**Explanation:** 18 is the smallest 2-digit number with digit sum 9.
