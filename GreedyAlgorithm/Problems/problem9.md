# Problem 9: Jump Game

## Problem Statement
Given an array of non-negative integers `nums`, you are initially positioned at the first index. Each element represents your maximum jump length from that position. Determine if you can reach the last index.

## Input Format
- A list of integers `nums`.

## Output Format
- A boolean — `True` if you can reach the last index, `False` otherwise.

## Constraints
- 1 <= len(nums) <= 10^4
- 0 <= nums[i] <= 10^5

## Example
**Input:** nums = [2, 3, 1, 1, 4]  
**Output:** True  
**Explanation:** Jump 1 step from index 0 to 1, then 3 steps to the last index.
