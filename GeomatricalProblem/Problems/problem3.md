# Problem 3: Convex Hull (Graham Scan)

## Problem Statement
Given a set of points, find the convex hull using Graham Scan algorithm.

## Input Format
- A list of points (x,y).

## Output Format
- A list of points forming the convex hull.

## Constraints
- 3 <= n <= 10^5

## Example
**Input:** points = [(0,3),(1,1),(2,2),(4,4),(0,0),(1,2),(3,1),(3,3)]  
**Output:** [(0,0),(3,1),(4,4),(0,3)]
