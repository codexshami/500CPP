# Problem 6: Line Intersection

## Problem Statement
Given two line segments, determine if they intersect and find the intersection point.

## Input Format
- Four points defining two line segments.

## Output Format
- Intersection point or None.

## Constraints
- -10^4 <= x,y <= 10^4

## Example
**Input:** Lines: (1,1)-(4,4) and (1,8)-(2,4)  
**Output:** Intersection point
