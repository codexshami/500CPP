# Problem 20: Maximum Points on a Line

## Problem Statement
Given a set of points, find the maximum number of points that lie on the same line.

## Input Format
- A list of points.

## Output Format
- An integer.

## Constraints
- 1 <= n <= 300

## Example
**Input:** points = [(1,1),(2,2),(3,3),(1,2)]  
**Output:** 3
