# Problem 8: Polygon Area (Shoelace Formula)

## Problem Statement
Compute the area of a polygon given its vertices using the Shoelace formula.

## Input Format
- A list of vertices in order.

## Output Format
- A float — area.

## Constraints
- 3 <= n <= 10^4

## Example
**Input:** vertices = [(0,0),(4,0),(4,3),(0,3)]  
**Output:** 12.0
