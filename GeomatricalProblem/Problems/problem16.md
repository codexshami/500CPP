# Problem 16: Minimum Enclosing Circle

## Problem Statement
Find the smallest circle that encloses all given points.

## Input Format
- A list of points.

## Output Format
- Center and radius of the circle.

## Constraints
- 1 <= n <= 10^5

## Example
**Input:** points = [(0,0),(1,0),(0,1)]  
**Output:** Center≈(0.5,0.5), r≈0.707
