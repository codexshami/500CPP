# Problem 10: Closest Pair of Points (Brute)

## Problem Statement
Find the closest pair of points in O(n²) time.

## Input Format
- A list of points.

## Output Format
- A float — minimum distance.

## Constraints
- 2 <= n <= 10^4

## Example
**Input:** points = [(1,1),(2,2),(3,3),(1,2)]  
**Output:** 1.0
