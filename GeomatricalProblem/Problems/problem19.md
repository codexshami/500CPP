# Problem 19: Lattice Points on Line Segment

## Problem Statement
Count the number of lattice (integer) points on a line segment between two points.

## Input Format
- Two points (x1,y1) and (x2,y2).

## Output Format
- An integer.

## Constraints
- -10^6 <= coordinates <= 10^6

## Example
**Input:** P1=(1,1), P2=(5,5)  
**Output:** 5
