# Problem 7: Circle-Line Intersection

## Problem Statement
Find the intersection points of a line and a circle.

## Input Format
- Circle center (cx,cy), radius r, line defined by two points.

## Output Format
- List of intersection points.

## Constraints
- -10^4 <= coordinates <= 10^4

## Example
**Input:** Circle:(0,0,5), Line:(0,0)-(5,0)  
**Output:** [(5,0),(-5,0)]
