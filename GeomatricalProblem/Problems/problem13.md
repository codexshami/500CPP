# Problem 13: Rotate Point Around Origin

## Problem Statement
Rotate a point by a given angle θ (in degrees) around the origin.

## Input Format
- A point (x,y) and angle θ.

## Output Format
- Rotated point (x',y').

## Constraints
- -10^4 <= x,y <= 10^4

## Example
**Input:** Point=(1,0), θ=90°  
**Output:** (0,1)
