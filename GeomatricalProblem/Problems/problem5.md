# Problem 5: Distance Between Two Points

## Problem Statement
Compute the Euclidean distance between two points in 2D space.

## Input Format
- Two points (x1,y1) and (x2,y2).

## Output Format
- A float.

## Constraints
- -10^6 <= x,y <= 10^6

## Example
**Input:** P1=(1,2), P2=(4,6)  
**Output:** 5.0
