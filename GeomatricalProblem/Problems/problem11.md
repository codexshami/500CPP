# Problem 11: Check if Point is on Line Segment

## Problem Statement
Given a line segment AB and point P, check if P lies on segment AB.

## Input Format
- Three points A, B, P.

## Output Format
- A boolean.

## Constraints
- -10^4 <= coordinates <= 10^4

## Example
**Input:** A=(0,0), B=(4,4), P=(2,2)  
**Output:** True
