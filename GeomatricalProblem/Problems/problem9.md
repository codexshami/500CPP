# Problem 9: Point in Polygon (Ray Casting)

## Problem Statement
Determine if a point lies inside a polygon using the ray casting algorithm.

## Input Format
- A polygon (list of vertices) and a point.

## Output Format
- A boolean.

## Constraints
- 3 <= vertices <= 10^4

## Example
**Input:** Polygon: [(0,0),(10,0),(10,10),(0,10)], P=(5,5)  
**Output:** True
