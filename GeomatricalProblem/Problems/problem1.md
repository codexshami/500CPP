# Problem 1: Point in Triangle

## Problem Statement
Given three vertices of a triangle and a point P, check if P lies inside the triangle.

## Input Format
- Four points (x,y).

## Output Format
- A boolean.

## Constraints
- Coordinates are integers, -10^4 <= x,y <= 10^4

## Example
**Input:** Triangle: (0,0),(10,30),(20,0), P=(10,15)  
**Output:** True
