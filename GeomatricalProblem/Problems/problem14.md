# Problem 14: Check Rectangle Overlap

## Problem Statement
Given two axis-aligned rectangles, check if they overlap.

## Input Format
- Two rectangles as [x1,y1,x2,y2].

## Output Format
- A boolean.

## Constraints
- -10^9 <= coordinates <= 10^9

## Example
**Input:** R1=[0,0,2,2], R2=[1,1,3,3]  
**Output:** True
