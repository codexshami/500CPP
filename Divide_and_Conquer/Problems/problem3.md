# Problem 3: Binary Search

## Problem Statement
Implement binary search to find the index of a target value in a sorted array. Return -1 if not found.

## Input Format
- A sorted list of integers `arr`.
- An integer `target`.

## Output Format
- An integer — index of target, or -1.

## Constraints
- 1 <= len(arr) <= 10^5
- -10^4 <= arr[i], target <= 10^4

## Example
**Input:** arr = [-1, 0, 3, 5, 9, 12], target = 9  
**Output:** 4
