# Problem 9: Median of Two Sorted Arrays

## Problem Statement
Given two sorted arrays `nums1` and `nums2`, return the median of the combined sorted array. The overall run time complexity should be O(log(m+n)).

## Input Format
- Two sorted lists `nums1` and `nums2`.

## Output Format
- A float — the median.

## Constraints
- 0 <= len(nums1), len(nums2) <= 1000
- -10^6 <= nums1[i], nums2[i] <= 10^6

## Example
**Input:** nums1 = [1, 3], nums2 = [2]  
**Output:** 2.0
