# Problem 12: Find Peak Element

## Problem Statement
A peak element is an element that is strictly greater than its neighbors. Given an array, find a peak element and return its index using O(log n) time.

## Input Format
- A list of integers `nums`.

## Output Format
- An integer — index of any peak element.

## Constraints
- 1 <= len(nums) <= 1000
- nums[i] != nums[i+1] for adjacent elements.

## Example
**Input:** nums = [1, 2, 3, 1]  
**Output:** 2
