# Problem 4: Count Inversions

## Problem Statement
Given an array, count the number of inversions. An inversion is a pair (i, j) where i < j and arr[i] > arr[j].

## Input Format
- A list of integers `arr`.

## Output Format
- An integer — count of inversions.

## Constraints
- 1 <= len(arr) <= 10^5
- 1 <= arr[i] <= 10^9

## Example
**Input:** arr = [2, 4, 1, 3, 5]  
**Output:** 3  
**Explanation:** Inversions: (2,1), (4,1), (4,3).
