# Problem 20: Sort a Linked List using Merge Sort

## Problem Statement
Sort a linked list in O(n log n) time and O(1) space using merge sort.

## Input Format
- Head of a singly linked list.

## Output Format
- Head of the sorted linked list.

## Constraints
- 0 <= number of nodes <= 5 * 10^4
- -10^5 <= Node.val <= 10^5

## Example
**Input:** head = [4, 2, 1, 3]  
**Output:** [1, 2, 3, 4]
