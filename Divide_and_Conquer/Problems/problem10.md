# Problem 10: Kth Largest Element

## Problem Statement
Find the kth largest element in an unsorted array using the QuickSelect algorithm (divide and conquer).

## Input Format
- A list of integers `nums`.
- An integer `k`.

## Output Format
- An integer — the kth largest element.

## Constraints
- 1 <= k <= len(nums) <= 10^5
- -10^4 <= nums[i] <= 10^4

## Example
**Input:** nums = [3,2,1,5,6,4], k = 2  
**Output:** 5
