# Problem 13: Majority Element

## Problem Statement
Given an array of size n, find the majority element (appears more than n/2 times) using divide and conquer.

## Input Format
- A list of integers `nums`.

## Output Format
- An integer — the majority element.

## Constraints
- 1 <= len(nums) <= 5 * 10^4
- The majority element always exists.

## Example
**Input:** nums = [2, 2, 1, 1, 1, 2, 2]  
**Output:** 2
