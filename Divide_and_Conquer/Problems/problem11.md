# Problem 11: Search in Rotated Sorted Array

## Problem Statement
Given a sorted array that has been rotated at an unknown pivot, search for a target value. Return its index or -1 if not found.

## Input Format
- A list of integers `nums` (rotated sorted).
- An integer `target`.

## Output Format
- An integer — index of target or -1.

## Constraints
- 1 <= len(nums) <= 5000
- All values are unique.

## Example
**Input:** nums = [4,5,6,7,0,1,2], target = 0  
**Output:** 4
