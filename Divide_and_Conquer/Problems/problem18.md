# Problem 18: Longest Common Prefix using Divide and Conquer

## Problem Statement
Find the longest common prefix string amongst an array of strings using a divide and conquer approach.

## Input Format
- A list of strings `strs`.

## Output Format
- A string — the longest common prefix.

## Constraints
- 1 <= len(strs) <= 200
- 0 <= len(strs[i]) <= 200

## Example
**Input:** strs = ["flower", "flow", "flight"]  
**Output:** "fl"
