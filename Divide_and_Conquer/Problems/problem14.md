# Problem 14: Count of Smaller Numbers After Self

## Problem Statement
Given an integer array `nums`, return a list where each element is the count of numbers to its right that are smaller than it. Use merge sort modification.

## Input Format
- A list of integers `nums`.

## Output Format
- A list of integers — counts.

## Constraints
- 1 <= len(nums) <= 10^5
- -10^4 <= nums[i] <= 10^4

## Example
**Input:** nums = [5, 2, 6, 1]  
**Output:** [2, 1, 1, 0]
