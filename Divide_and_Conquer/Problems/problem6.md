# Problem 6: Closest Pair of Points

## Problem Statement
Given `n` points in a 2D plane, find the pair of points with the smallest Euclidean distance.

## Input Format
- A list of tuples `points` where each tuple `(x, y)` represents a point.

## Output Format
- A float — the minimum distance between any two points.

## Constraints
- 2 <= n <= 10^5
- -10^6 <= x, y <= 10^6

## Example
**Input:** points = [(2,3), (12,30), (40,50), (5,1), (12,10), (3,4)]  
**Output:** 1.414 (between (2,3) and (3,4))
