# Problem 15: Karatsuba Multiplication

## Problem Statement
Implement the Karatsuba algorithm for multiplying two large numbers efficiently.

## Input Format
- Two integers `x` and `y`.

## Output Format
- An integer — the product.

## Constraints
- 0 <= x, y <= 10^18

## Example
**Input:** x = 1234, y = 5678  
**Output:** 7006652
