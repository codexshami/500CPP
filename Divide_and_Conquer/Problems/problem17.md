# Problem 17: Different Ways to Add Parentheses

## Problem Statement
Given a string `expression` of numbers and operators (+, -, *), compute all possible results from all different ways to group numbers and operators using parentheses.

## Input Format
- A string `expression`.

## Output Format
- A list of integers — all possible results.

## Constraints
- 1 <= len(expression) <= 20

## Example
**Input:** expression = "2*3-4*5"  
**Output:** [-34, -14, -10, -10, 10]
