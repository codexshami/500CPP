# Problem 5: Maximum Subarray (Kadane's Divide & Conquer)

## Problem Statement
Find the contiguous subarray with the largest sum using divide and conquer approach.

## Input Format
- A list of integers `nums`.

## Output Format
- An integer — maximum subarray sum.

## Constraints
- 1 <= len(nums) <= 10^5
- -10^4 <= nums[i] <= 10^4

## Example
**Input:** nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]  
**Output:** 6  
**Explanation:** Subarray [4, -1, 2, 1] has sum 6.
