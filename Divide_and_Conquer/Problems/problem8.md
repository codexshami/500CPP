# Problem 8: Power Function (x^n)

## Problem Statement
Implement `pow(x, n)`, which calculates `x` raised to the power `n` using divide and conquer.

## Input Format
- A float `x` and an integer `n`.

## Output Format
- A float — x raised to power n.

## Constraints
- -100.0 < x < 100.0
- -2^31 <= n <= 2^31 - 1

## Example
**Input:** x = 2.0, n = 10  
**Output:** 1024.0
