# Problem 7: Strassen's Matrix Multiplication

## Problem Statement
Implement Strassen's algorithm for matrix multiplication of two n×n matrices.

## Input Format
- Two 2D lists `A` and `B` representing n×n matrices.

## Output Format
- A 2D list representing the product matrix.

## Constraints
- n is a power of 2
- 1 <= n <= 128

## Example
**Input:** A = [[1,2],[3,4]], B = [[5,6],[7,8]]  
**Output:** [[19,22],[43,50]]
