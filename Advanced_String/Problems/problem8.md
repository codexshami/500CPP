# Problem 8: Regular Expression Matching

## Problem Statement
Implement regular expression matching with '.' and '*' support.

## Input Format
- A string `s` and pattern `p`.

## Output Format
- A boolean.

## Constraints
- 1 <= len(s) <= 20, 1 <= len(p) <= 30

## Example
**Input:** s="aab", p="c*a*b"  
**Output:** True
