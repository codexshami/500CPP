# Problem 18: Palindrome Partitioning

## Problem Statement
Partition a string such that every substring is a palindrome. Return all possible partitions.

## Input Format
- A string `s`.

## Output Format
- A list of lists.

## Constraints
- 1 <= len(s) <= 16

## Example
**Input:** s="aab"  
**Output:** [["a","a","b"],["aa","b"]]
