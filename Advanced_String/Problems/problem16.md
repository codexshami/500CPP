# Problem 16: Group Anagrams

## Problem Statement
Given an array of strings, group the anagrams together.

## Input Format
- A list of strings.

## Output Format
- A list of groups.

## Constraints
- 1 <= len(strs) <= 10^4

## Example
**Input:** strs=["eat","tea","tan","ate","nat","bat"]  
**Output:** [["eat","tea","ate"],["tan","nat"],["bat"]]
