# Problem 15: Word Break

## Problem Statement
Given a string `s` and a dictionary of words, determine if `s` can be segmented into dictionary words.

## Input Format
- A string and a list of words.

## Output Format
- A boolean.

## Constraints
- 1 <= len(s) <= 300

## Example
**Input:** s="leetcode", wordDict=["leet","code"]  
**Output:** True
