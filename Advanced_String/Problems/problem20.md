# Problem 20: Text Justification

## Problem Statement
Given words and a max width, format text with full justification.

## Input Format
- A list of words and maxWidth.

## Output Format
- A list of justified strings.

## Constraints
- 1 <= len(words) <= 300, 1 <= maxWidth <= 100

## Example
**Input:** words=["This","is","an","example"], maxWidth=16  
**Output:** ["This    is    an", "example         "]
