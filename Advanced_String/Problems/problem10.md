# Problem 10: Shortest Palindrome

## Problem Statement
Find the shortest palindrome by adding characters in front of a given string.

## Input Format
- A string `s`.

## Output Format
- The shortest palindrome.

## Constraints
- 0 <= len(s) <= 5*10^4

## Example
**Input:** s="aacecaaa"  
**Output:** "aaacecaaa"
