# Problem 5: Power Modular Exponentiation

## Problem Statement
Compute `(base^exp) % mod` efficiently using modular exponentiation.

## Input Format
- Three integers `base`, `exp`, `mod`.

## Output Format
- An integer.

## Constraints
- 1 <= base, exp, mod <= 10^9

## Example
**Input:** base=2, exp=10, mod=1000  
**Output:** 24
