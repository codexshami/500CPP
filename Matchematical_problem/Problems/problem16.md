# Problem 16: Happy Number

## Problem Statement
Determine if a number is happy. Replace the number by the sum of squares of its digits repeatedly. If it reaches 1, it's happy.

## Input Format
- An integer `n`.

## Output Format
- A boolean.

## Constraints
- 1 <= n <= 2^31 - 1

## Example
**Input:** n = 19  
**Output:** True
