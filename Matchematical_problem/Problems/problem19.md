# Problem 19: Armstrong Number

## Problem Statement
Check if a number is an Armstrong number (sum of cubes of digits equals the number for 3-digit numbers, generalized for k-digit).

## Input Format
- An integer `n`.

## Output Format
- A boolean.

## Constraints
- 1 <= n <= 10^9

## Example
**Input:** n = 153  
**Output:** True (1³+5³+3³=153)
