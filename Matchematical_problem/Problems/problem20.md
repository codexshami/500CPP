# Problem 20: Excel Sheet Column Number

## Problem Statement
Convert an Excel column title to its corresponding column number. A=1, B=2, ..., Z=26, AA=27.

## Input Format
- A string `columnTitle`.

## Output Format
- An integer.

## Constraints
- 1 <= len(columnTitle) <= 7

## Example
**Input:** columnTitle = 'AB'  
**Output:** 28
