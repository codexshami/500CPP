# Problem 1: GCD and LCM

## Problem Statement
Given two integers `a` and `b`, find their Greatest Common Divisor (GCD) and Least Common Multiple (LCM).

## Input Format
- Two integers `a` and `b`.

## Output Format
- A tuple `(gcd, lcm)`.

## Constraints
- 1 <= a, b <= 10^9

## Example
**Input:** a = 12, b = 18  
**Output:** (6, 36)
