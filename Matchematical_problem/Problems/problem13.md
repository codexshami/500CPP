# Problem 13: Euler's Totient Function

## Problem Statement
Compute Euler's totient function φ(n) — count of integers from 1 to n that are coprime with n.

## Input Format
- An integer `n`.

## Output Format
- An integer.

## Constraints
- 1 <= n <= 10^6

## Example
**Input:** n = 10  
**Output:** 4
