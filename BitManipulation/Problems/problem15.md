# Problem 15: Toggle the i-th Bit

## Problem Statement
Given an integer `n` and a position `i`, toggle the i-th bit (0-indexed from right). If the bit is 1, set to 0, and vice versa.

## Input Format
- An integer `n`.
- An integer `i`.

## Output Format
- An integer with the i-th bit toggled.

## Constraints
- 0 <= n <= 2^31 - 1
- 0 <= i <= 30

## Example
**Input:** n = 5 (binary: 101), i = 1  
**Output:** 7 (binary: 111)
