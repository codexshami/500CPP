# Problem 20: Maximum XOR of Two Numbers in an Array

## Problem Statement
Given an integer array `nums`, return the maximum result of `nums[i] XOR nums[j]` where `0 <= i <= j < n`.

## Input Format
- A list of integers `nums`.

## Output Format
- An integer — maximum XOR value.

## Constraints
- 1 <= len(nums) <= 2 * 10^5
- 0 <= nums[i] <= 2^31 - 1

## Example
**Input:** nums = [3, 10, 5, 25, 2, 8]  
**Output:** 28  
**Explanation:** Maximum XOR is 5 XOR 25 = 28.
