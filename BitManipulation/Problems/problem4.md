# Problem 4: Counting Bits

## Problem Statement
Given an integer `n`, return an array `ans` of length `n + 1` such that for each `i` (0 <= i <= n), `ans[i]` is the number of 1s in the binary representation of `i`.

## Input Format
- An integer `n`.

## Output Format
- A list of integers of length `n + 1`.

## Constraints
- 0 <= n <= 10^5

## Example
**Input:** n = 5  
**Output:** [0, 1, 1, 2, 1, 2]
