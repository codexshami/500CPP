# Problem 14: Check if i-th Bit is Set

## Problem Statement
Given an integer `n` and a bit position `i`, check whether the i-th bit (0-indexed from right) is set (1) or not (0).

## Input Format
- An integer `n`.
- An integer `i` — bit position.

## Output Format
- A boolean.

## Constraints
- 0 <= n <= 2^31 - 1
- 0 <= i <= 30

## Example
**Input:** n = 5 (binary: 101), i = 2  
**Output:** True
