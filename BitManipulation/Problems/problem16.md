# Problem 16: Count Total Set Bits from 1 to N

## Problem Statement
Given a positive integer `n`, count the total number of set bits in binary representation of all numbers from 1 to n.

## Input Format
- An integer `n`.

## Output Format
- An integer — total set bits.

## Constraints
- 1 <= n <= 10^8

## Example
**Input:** n = 4  
**Output:** 5  
**Explanation:** 1(1) + 1(1) + 2(10) + 1(1) + 1(1) → total = 5 set bits for 1,2,3,4
