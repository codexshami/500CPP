# Problem 3: Power of Two

## Problem Statement
Given an integer `n`, return `True` if it is a power of two, otherwise return `False`.

## Input Format
- An integer `n`.

## Output Format
- A boolean.

## Constraints
- -2^31 <= n <= 2^31 - 1

## Example
**Input:** n = 16  
**Output:** True  
**Explanation:** 2^4 = 16
