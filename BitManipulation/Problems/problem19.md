# Problem 19: Find the Rightmost Set Bit

## Problem Statement
Given a positive integer `n`, find the position (1-indexed from right) of the rightmost set bit.

## Input Format
- An integer `n`.

## Output Format
- An integer — position of rightmost set bit.

## Constraints
- 1 <= n <= 2^31 - 1

## Example
**Input:** n = 12 (binary: 1100)  
**Output:** 3
