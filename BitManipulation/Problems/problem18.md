# Problem 18: Minimum Flips to Make a OR b Equal to c

## Problem Statement
Given three positive integers `a`, `b`, and `c`, return the minimum number of bit flips required to make `a OR b == c`.

## Input Format
- Three integers `a`, `b`, `c`.

## Output Format
- An integer — minimum flips.

## Constraints
- 1 <= a, b, c <= 10^9

## Example
**Input:** a = 2, b = 6, c = 5  
**Output:** 3
