# Problem 13: Swap Two Numbers Without Temp Variable

## Problem Statement
Given two integers `a` and `b`, swap them without using a temporary variable, using XOR bit manipulation.

## Input Format
- Two integers `a` and `b`.

## Output Format
- Two integers (swapped values).

## Constraints
- -10^9 <= a, b <= 10^9

## Example
**Input:** a = 5, b = 10  
**Output:** a = 10, b = 5
