# Problem 7: Hamming Distance

## Problem Statement
The Hamming distance between two integers is the number of positions at which the corresponding bits are different. Given two integers `x` and `y`, return the Hamming distance between them.

## Input Format
- Two integers `x` and `y`.

## Output Format
- An integer — the Hamming distance.

## Constraints
- 0 <= x, y <= 2^31 - 1

## Example
**Input:** x = 1, y = 4  
**Output:** 2  
**Explanation:** 1 (001) and 4 (100) differ in 2 bit positions.
