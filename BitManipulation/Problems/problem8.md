# Problem 8: Sum of Two Integers Without Arithmetic Operators

## Problem Statement
Given two integers `a` and `b`, return their sum without using the `+` or `-` operators.

## Input Format
- Two integers `a` and `b`.

## Output Format
- An integer — the sum.

## Constraints
- -1000 <= a, b <= 1000

## Example
**Input:** a = 1, b = 2  
**Output:** 3
