# Problem 10: Single Number III

## Problem Statement
Given an integer array `nums` where exactly two elements appear only once and all other elements appear exactly twice, find the two elements that appear only once. Return them in any order.

## Input Format
- A list of integers `nums`.

## Output Format
- A list of two integers.

## Constraints
- 2 <= len(nums) <= 3 * 10^4
- -2^31 <= nums[i] <= 2^31 - 1

## Example
**Input:** nums = [1, 2, 1, 3, 2, 5]  
**Output:** [3, 5]
