# Problem 11: Bitwise AND of Numbers Range

## Problem Statement
Given two integers `left` and `right` representing a range `[left, right]`, return the bitwise AND of all numbers in this range.

## Input Format
- Two integers `left` and `right`.

## Output Format
- An integer.

## Constraints
- 0 <= left <= right <= 2^31 - 1

## Example
**Input:** left = 5, right = 7  
**Output:** 4
