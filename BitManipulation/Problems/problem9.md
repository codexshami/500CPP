# Problem 9: Single Number II

## Problem Statement
Given an integer array `nums` where every element appears exactly three times except for one element which appears exactly once. Find the single element.

## Input Format
- A list of integers `nums`.

## Output Format
- An integer — the element appearing once.

## Constraints
- 1 <= len(nums) <= 3 * 10^4
- -2^31 <= nums[i] <= 2^31 - 1

## Example
**Input:** nums = [2, 2, 3, 2]  
**Output:** 3
