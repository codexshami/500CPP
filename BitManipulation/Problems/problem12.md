# Problem 12: Power Set using Bits

## Problem Statement
Given a set of distinct integers `nums`, return all possible subsets (the power set) using bit manipulation.

## Input Format
- A list of distinct integers `nums`.

## Output Format
- A list of lists representing all subsets.

## Constraints
- 1 <= len(nums) <= 10
- -10 <= nums[i] <= 10

## Example
**Input:** nums = [1, 2, 3]  
**Output:** [[], [1], [2], [1,2], [3], [1,3], [2,3], [1,2,3]]
