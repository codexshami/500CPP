# Problem 17: Find Two Non-Repeating Elements

## Problem Statement
Given an array where all elements appear twice except two, find those two non-repeating elements using bit manipulation.

## Input Format
- A list of integers `nums`.

## Output Format
- A list of two integers.

## Constraints
- 2 <= len(nums) <= 3 * 10^4

## Example
**Input:** nums = [2, 4, 7, 9, 2, 4]  
**Output:** [7, 9]
