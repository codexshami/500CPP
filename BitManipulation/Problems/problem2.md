# Problem 2: Number of 1 Bits (Hamming Weight)

## Problem Statement
Write a function that takes the binary representation of a positive integer and returns the number of set bits (1s) it has.

## Input Format
- An integer `n`.

## Output Format
- An integer — count of set bits.

## Constraints
- 0 <= n <= 2^31 - 1

## Example
**Input:** n = 11 (binary: 1011)  
**Output:** 3
