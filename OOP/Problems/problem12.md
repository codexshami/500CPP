# Problem 12: Implement Decorator Pattern

## Problem Statement
Design a coffee ordering system where extras (milk, sugar) are decorators.

## Input Format
- Base item + decorators.

## Output Format
- Description and total cost.

## Constraints
- 1 <= decorators <= 5

## Example
**Input:** Coffee() + Milk() + Sugar()  
**Output:** Coffee, Milk, Sugar: $3.50
