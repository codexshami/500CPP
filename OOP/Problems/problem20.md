# Problem 20: Design a Social Media Post System

## Problem Statement
Create User, Post, and Comment classes with like and reply functionality.

## Input Format
- Social actions.

## Output Format
- Post details.

## Constraints
- 1 <= likes <= 10^6

## Example
**Input:** post.like(user), post.comment(user, text)  
**Output:** 1 like, 1 comment
