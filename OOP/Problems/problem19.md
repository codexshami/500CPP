# Problem 19: Design a Queue using Two Stacks

## Problem Statement
Implement a Queue using two Stack objects (OOP composition).

## Input Format
- Queue operations: enqueue, dequeue.

## Output Format
- Values from dequeue.

## Constraints
- 1 <= operations <= 10^4

## Example
**Input:** enqueue(1), enqueue(2), dequeue()  
**Output:** 1
