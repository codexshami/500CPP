# Problem 15: Design a Tic-Tac-Toe Game

## Problem Statement
Implement a Tic-Tac-Toe game with a Board class and Player classes.

## Input Format
- Player moves.

## Output Format
- Game state (win/draw/continue).

## Constraints
- 3x3 board

## Example
**Input:** move(0,0,'X'), move(1,1,'O')  
**Output:** Continue
